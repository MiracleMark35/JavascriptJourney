# What is Node JS
it is Scalability and performance - it can handles large application seamlessly.
it is the same language from front-end and backend it is also cost effective due to same language no need to spend development to other languages.
it is real time application means it can handle chat applications and online gaming.
it can create apps in windows linux.

# Meaning 
Node js is runtime environment that allows javascript runs on a server.
it can execute javascript code outside a website, like in a terminal.

# runtime environment
- is the set-up that allows a program to run.
  
# server side
- it means code run in a server, handles logic, hidden from users.

# client side 
- runs in the users browser, handles UI, and visible to users.

# Node js uses
- is used by big companies like netflix, uber  and paypal.
- it offers robust and its scalability.

# NPM 
-stands for Node Package Manager.
- it can install libraries.
- manages packages.
- run project scripts.

# Package
- instead of writing from scratch u can install packages using npm.
- it is a resusable javascript code.
- examples, JSONTOKEN, express, nodemon.

| Command               | Purpose                  |
| --------------------- | ------------------------ |
| `npm init`            | Create package.json      |
| `npm install`         | Install all dependencies |
| `npm install <pkg>`   | Install a package        |
| `npm uninstall <pkg>` | Remove a package         |
| `npm run <script>`    | Run scripts              |
| `npm update`          | Update packages          |

# Dependencies
- are external libraires or packages that your project needs to work.

# Modules
- are resusable block of code that you can be imported in to other files.
- Built in modules (comes from node.js).
- Local or User defined (created by the user).

# Express
- is a popular website framework for Node.js that simplifies process of building web applications.
- basically it is a framework that can create api and servers easily.
